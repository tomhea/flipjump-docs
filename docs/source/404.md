## Maybe you wanted

- **A macro?** Try the [alphabetical macro index](stl/all_macros.md).
- **A specific STL file?** All files are under [`stl/`](stl/index.md) — file pages live at e.g. `/stl/bit/math.html`, `/stl/runlib.html`.
- **A specific macro page?** Macros live nested under their file — e.g. `bit.add` arity 3 → `/stl/bit/math/add--3.html`.
- **A language reference page?** Check [Language Reference](language/index.md).
- **A tutorial?** Start with [Getting Started](getting-started/index.md) or the [Cookbook](cookbook/index.md).
- **A specific concept?** The [Glossary](reference/glossary.md) defines every FJ-specific term.

## Reporting a broken link

If you got here by clicking a link inside this docs site, that's a bug — [open an issue on `flipjump-docs`](https://github.com/tomhea/flipjump-docs/issues) and mention which page you were on.

If you got here from an external link or a search engine result, that source may be referencing an old URL — the site was restructured in v1.1 to put file pages at `/stl/<path>.html` instead of `/stl/_generated/file--*.html`. Updating the linking source is the right fix.
