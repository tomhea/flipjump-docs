## In a hurry?

Open the [FlipJump IDE](https://fj.tomhe.app) in your browser — it has the compiler and runtime built-in, no local install required. Pick the "Hello World" example from the file menu, click Run, and you're done.

When you're ready to use it locally, [install the `fj` toolchain](install.md) and follow the [Hello World walkthrough](hello-world.md).
